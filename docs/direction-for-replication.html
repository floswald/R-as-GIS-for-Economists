Direction for replication

Running the codes in this book sometimes involves reading datasets from a disk (many of the datasets will be downloaded from within R). All the datasets that will be imported are available here. In this book, the path to files to import is set relative to my own working directory (which is hidden). To run the codes without having to mess with paths to files, follow these steps:7

  • set a folder as the working directory using setwd()
  • create a folder called “Data” inside the folder designated as the working directory
  • download the pertinent datasets from here and put them in the “Data” folder

Some codes are too long or tangential to display on the lecture note. Those codes are written in files separately for each chapter. For example, for Chapter 1, it is named Chap_1_Demonstration.R. All these files can be downloaded from here. Run them before you start replicating.
